## Instructions

`mix deps.get` & run the following command to generate a `yourstring.png` inside of the `/identicons` folder

**Note**: Replace "yourstring" with any string value for a unique identicon

```
iex> Identicon.main("yourstring")    
      :ok
```

Enjoy!
